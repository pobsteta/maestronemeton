# Telecharger le toy dataset FLAIR-HUB

Telecharge un petit jeu de donnees d'exemple pour tester le pipeline.

## Usage

``` r
download_flair_toy(data_dir = "data/flair_hub_toy", token = NULL)
```

## Arguments

- data_dir:

  Repertoire de destination

- token:

  Token HuggingFace (optionnel)

## Value

Chemin du repertoire contenant les donnees
