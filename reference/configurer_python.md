# Configurer l'environnement Python pour MAESTRO

Configure reticulate pour utiliser l'environnement conda `maestro` et
verifie que tous les modules requis sont disponibles. Pattern identique
a `setup_python()` de flair_hub_nemeton.

## Usage

``` r
configurer_python(envname = CONDA_ENV)
```

## Arguments

- envname:

  Nom de l'environnement conda (defaut: "maestro")

## Value

Invisible NULL
