# Scanner les fichiers FLAIR-HUB disponibles localement

Explore un repertoire pour detecter les patches FLAIR-HUB et leurs
modalites.

## Usage

``` r
scan_flair_files(data_dir = "data/flair_hub")
```

## Arguments

- data_dir:

  Repertoire contenant les donnees FLAIR-HUB

## Value

Liste nommee par modalite, chacune contenant les chemins des fichiers
