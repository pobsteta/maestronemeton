# Telecharger un fichier depuis un dataset HuggingFace

Telecharger un fichier depuis un dataset HuggingFace

## Usage

``` r
hf_download_file(repo_id, filename, dest_dir, token = NULL)
```

## Arguments

- repo_id:

  Identifiant du repository (ex: "IGNF/FLAIR-HUB")

- filename:

  Nom du fichier dans le repo

- dest_dir:

  Repertoire de destination local

- token:

  Token HuggingFace (optionnel)

## Value

Chemin du fichier telecharge
